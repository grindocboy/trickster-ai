# Trickster AI

Learn Spades by actually playing.

Trickster AI is an AI-powered Spades trainer that teaches players through gameplay instead of long tutorials.

## Features

✅ AI opponents

✅ AI partner

✅ Beginner to Expert difficulty

✅ Coaching Mode

✅ Bid recommendations

✅ Trick analysis

✅ Practice hands

✅ Statistics

✅ Mobile friendly
